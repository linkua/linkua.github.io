ESP8266

ESP8266是一款低成本、低功耗、高度集成的Wi-Fi芯片，其主要参数包括：12

主频：80 MHz (单核)
内存：80 KB RAM + 4 MB Flash
Wi-Fi标准：802.11 b/g/n 不支持蓝牙
工作电压：3.0V - 3.6V
工作温度：-40℃ ~ 125℃
支持模式：STA/AP/AP+STA
支持协议：TCP/UDP
支持连接：多连接
支持升级：OTA升级
此外，ESP8266还具备以下特性：

无线参数：支持802.11 b/g/n标准，频率范围2.4GHz至2.5GHz，具有不同的发射功率和接收灵敏度，以满足不同标准下的通信需求。
存储系统：配备有32Mb(4M)的Flash存储器和80KB的SRAM，Flash存储器分为几个部分，包括Bootloader等。
处理器和性能：内置超低功耗的Tensilica L106 32位RISC处理器，最高时钟速度可达160MHz，支持RTOS和Wi-Fi协议栈，能够将高达80%的处理能力留给应用编程和开发。

ESP8266模块有多个型号和变种，例如ESP-01、ESP-12E、NodeMCU等，每种型号的功能和特点可能略有不同。

ESP32-C3发布已经有一段时间了，作为ESP8266的替代品，上架之初C3模组就保持了10元以下的友好价格，并在8266的基础上增加了蓝牙支持。
内建的USB-CDC支持，提供一个USB串口以及JTAG调试器。这表明我们可以跳过CH340、CP2102等USB转串口芯片直接对C3模组进行开发调试。

MCU 与 MPU

Posted on 2024-04-12 Edited on 2024-05-20 In MCU MPU

MCU 与 MPU

MCU的全称是Mirco Controller Unit，MPU的全称叫Micro Processor Unit。
微控制器（单片机，Cortex-M0/M3内核架构，没有内存管理单元MMU），早期8位8051系列主频不过10几MHz，flash 1-10k, ram 256/512字节， 还是12个周期执行一条指令。而随着ARM的32位MCU的出现，采用ARM的M系列的MCU也开始逐步扩大市场。以ST、NXP公司的产品为主要代表，stm32fl103, flash 32-128k, ram 10-48k。但是这些ARM系列MCU的主频一般也是在几十MHz到100多MHz的量级。
随着物理网发展，一些带有wifi/蓝牙的SOC,弥补MCU与MPU的空白，例如 ESP3 频率在240MHz, 520KB SRAM，448KB ROM，同时支持外部4MB-16 SPI Flash. 还有树莓派 pico rp2040,双核 Arm Cortex-M0 + @ 133MHz
芯片内置 264KB SRAM 和 2MB 的板载闪存。。

而微处理器（嵌入式，Cotex-A内核架构，主要指应用型处理器），其实就是微型化/集成化了的处理器。准确的说是微型化/集成化的“中央处理器”。MPU从一开始就定位了具有相当的处理和运算能力，一帮需要运行较大型的操作系统(如linux、opentwrt)来实现复杂的任务处理。因此这就决定了MPU应该具备比较高的主频，和较为强大的运算能力和内存管理单元MMU。MPU很早就演进到了32位处理器，现在更是开始大力普及64位。同时MPU也一直追求实现较高的主频。早期经典的Arm 9系列MPU频率就在200MHz-400MHz。现在手机上使用的高端MPU更是到达了3GHz，和主流的桌面处理器是一个级别了。和通用的桌面处理器一样，MPU现在也普遍“多核化”。

FREERTOS 常用于 MCU, LINUX常用于MPU, LINUX需要硬件支持MMU, LINUX的内核空间与应用程序空间分开,进程独享全部内存空间，而FREERTOS与任务同一空间（单进程），只做线程任务调度。例如路由器使用的就是MPU，通常运行openwrt,要求4MB flash和32MB RAM.

  3. VSCode clangd 插件本身是不能运行的，它还需要 clangd 的环境，所以接下来安装 clangd。
    在 Ubuntu 中最好不要使用 apt 安装，因为版本比较旧，建议下载后手动安装。
    解压后，
    将 bin 文件夹下的 clangd 移动到 /usr/bin 目录下，
    将 lib 文件夹下的所有移动到 /usr/local/lib 目录下。
    clangd --version查看版本。

F1C100s/F1C200s

  • F1C100s 的内核是 ARM926EJ-S ，ARMv5 架构，有 MMU，没有硬件 FPU，主频默认为 408MHz；
  • F1C100s 内置 32MB DDR1 内存；
  • F1C200s 是 F1C100s 的兄弟型号，其它特性都相同，内存容量翻倍为 64MB；
